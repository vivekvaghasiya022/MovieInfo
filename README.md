# movieinfo

A Flutter application for Movie/TV show's basic information like Ratings(IMDB, Rotten Tomatoes), Cast, Pilot, etc.

I have used OMDb API for get json data of movies and shows.
It's free for use.
http://www.omdbapi.com/
